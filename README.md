# Markdown Viewer Premium - Website 💎

Landing page and live demo for [Markdown Viewer Premium](https://github.com/chieund/markdown-previewer) — a beautiful, cross-platform Markdown previewer.

## Features

- Interactive live demo with real-time Markdown rendering
- Mermaid diagrams, Math/LaTeX, syntax highlighting
- Responsive glassmorphism UI

## Development

```bash
pnpm install
pnpm dev
```

## Build & Deploy

```bash
pnpm build
```

Automatically deployed to GitHub Pages on push to `main`.
